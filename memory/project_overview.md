Admin roles: SUPER_ADMIN (full access), BLOG_EDITOR (blog only), PLATFORM_EDITOR (platforms only).

Dark theme: bg #060910, cards #0f1629, accent yellow-400.

**Why:** Full-stack betting comparison site for Indian bettors.
**How to apply:** Follow existing dark theme and role-based access patterns when adding admin features.
